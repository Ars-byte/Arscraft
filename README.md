# Arscraft

Hola, soy Ars-Byte y hoy te presento mi tipo de minecraft hecho en python.

## Dependencias

pip install -r requirements.txt

Si usas Ubuntu/Debian/Mint:
sudo apt install python3-tk

Si usas Arch Linux/Manjaro:
sudo pacman -S tk

### Controles

AWSD: movimiento
click izquierdo: poner bloques
click derecho: quitar bloques
1/2/3/4: selector de bloques de la hotbar

### Ejecucion

Puedes ejecutarlo facilmente con los siguientes comandos:

python3 main.py

o si quieres ver el launcher 

python3 launcher_ars.py
